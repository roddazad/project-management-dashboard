# TaskFlow 🧠✨  
**A sleek and powerful Project Management Dashboard inspired by Trello.**

TaskFlow is a lightweight, intuitive task board designed to help users manage and visualize their workflow. Whether you're tracking personal goals or team projects, TaskFlow helps keep everything organized, prioritized, and on schedule — all within a clean, responsive UI.

---

## 🔥 Features

- 🔐 **Login / Signup system** (with localStorage session handling)
- 🧠 Personalized **dashboard greeting** for each user
- 📝 Add tasks with **title**, **priority**, and **due date**
- 📦 Organize tasks by column: **To Do**, **In Progress**, and **Done**
- 🧲 **Drag and drop** tasks between columns
- 🖌️ Color-coded **priority labels** (Low, Medium, High)
- ⚠️ **Due date awareness** (Overdue / Today / Upcoming)
- ✏️ Edit tasks
- 🗑️ Delete tasks
- 🔓 **Logout functionality**
- 🏠 Home and 📬 Contact links in dashboard navbar
- 💾 Data stored in **LocalStorage**
- 💬 **Contact form** (via [Formspree](https://formspree.io)) that emails submissions
- 💻 Fully **responsive design** with hover effects and animation
